# WorkHelper

WorkHelper は、Windows で使う業務補助アプリです。タスク、メモ、社員リスト、時間外・36協定の確認などを、1つのアプリで扱えます。

## はじめて使う方

1. ブラウザで **[Releases ページ](https://github.com/Akeytec/WorkHelper/releases)** を開きます。
2. 一番上の **最新バージョン** を開きます。
3. 初回だけ、**Assets** から **`WorkHelper-Certificate-Setup.zip`** をダウンロードして展開し、**`証明書を登録.bat`** を実行します。
4. **Assets** から **`WorkHelper-Setup-x.x.x.exe`**（例: `WorkHelper-Setup-2.0.4.exe`）をダウンロードします。
5. ダウンロードした `.exe` をダブルクリックしてインストールします。
6. インストールが終わったら、**スタートメニュー** から **WorkHelper** を起動します。

通常の利用に、Node.js や Git などの準備は不要です。証明書登録は、同じ PC では基本的に初回だけで大丈夫です。

ダウンロード中に表示される **`.crdownload`** ファイルは、まだ途中のファイルです。必ずダウンロード完了後の **`.zip`** または **`WorkHelper-Setup-x.x.x.exe`** を開いてください。

## 最初に知っておくこと

- データは自動で保存されます。
- 普段の操作はオフラインで使えます。
- 更新確認やインストーラーのダウンロードには、インターネット接続が必要です。
- Smart App Control 対策として、初回だけ WorkHelper 用の証明書登録が必要です。

## 主な使い方

### Home

よく使う機能への入口です。

Home の **マイタスク** の下に、アプリが最新かどうかも表示されます。

- **最新** と表示されている場合: `更新を確認` で再確認できます。
- **更新が必要** と表示されている場合: `アップデート` で更新できます。

### マイタスク

今日やることを登録できます。Home からも未完了タスクを確認できます。

### メモ

一時的なメモや作業記録を残せます。入力内容は自動で保存されます。

### 社員リスト

社員番号、氏名、所属などを管理できます。必要に応じて列や行を追加できます。

### 時間外・36協定

勤怠CSVを取り込み、社員リストや設定した上限と照合して確認できます。結果は Excel に出力できます。

### バックアップ

大切なデータを守るため、定期的に **すべてエクスポート（JSON）** を行ってください。

バックアップ画面では、保存先フォルダを指定することもできます。指定した場合は `workhelper-storage.json` が作成されます。

## 更新について

WorkHelper は起動時に自動で更新を確認します。手動で確認したい場合は、Home の `更新を確認` またはメニューの **ヘルプ > 更新を確認** を使ってください。

更新がある場合は、ダウンロード完了後にアプリの再起動で反映されます。

## 困ったとき

- インストールできない: `.crdownload` ではなく、ダウンロード完了後の `.zip` または `.exe` を開いているか確認してください。Smart App Control で止まる場合は、先に `WorkHelper-Certificate-Setup.zip` の `証明書を登録.bat` を実行してください。
- 更新できない: インターネット接続と、GitHub へのアクセスが許可されているか確認してください。
- データ移行が不安: 先にバックアップ画面で **すべてエクスポート（JSON）** を実行してください。

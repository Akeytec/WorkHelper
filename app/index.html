        社員リスト

        初期の列は「社員番号」「氏名」「所属」です。セルの編集はいつでも行え、自動で保存されます。 「行を追加」「列を追加」「上へ」「下へ」「列削除」「行削除」は編集モードをオンにしたときだけ表示されます。列は最低1列必要です。 テンプレート（CSV）は保存済みの列見出しに基づきます（列を増やしたあとにダウンロードすると列も増えます）。CSVインポートは1行目を列名として社員リストをまるごと置き換えます。

        時間外・36協定

        対応するCSV：マネーフォワード勤怠の法定外・休日労働時間レポート、または法定外時間（平日）などがある月次勤怠集計。UTF-8／Shift_JIS は自動で判読します。

        照合のしかた：CSVの従業員番号＝社員リストの社員番号。上限は36協定 値設定の所属別の値を、社員リストの所属と対応させます。

        1ヶ月の色：比較値が上限の設定％未満は緑（以内）、同％以上で未超過は黄（警告）、超過は赤。％は値設定で変えられます。

        複数月・年度：月次CSVを取り込み、履歴に登録すると一覧に反映されます（既定は直近5ヶ月の平均と、1月〜12月の年間合計）。細かい設定は下の集計期間パネル（年間の形は編集モードで変更）。

        一覧とExcel：編集モードで列の非表示・再表示（社員番号は常に表示）。Excelは、見えている列と判定どおりに出力。協定の数値を変えたら、ダイアログで保存してからエクスポートしてください。

        本機能は法的助言ではありません。

        月次勤怠集計のCSVを取り込みました。対象の年月を選び、履歴に取り込むと、複数月平均・年度累計の算出に使われます（このブラウザ内の保存領域（localStorage）に保存されます）。

        既定：複数月平均は直近5ヶ月の平均、年間（12ヶ月）の合計は1月〜12月（暦年）です。「開始月から12ヶ月」に切り替えると、例として4月始まりなら翌年3月までのように連続12ヶ月で集計します。年間の形・西暦年は一覧の編集モードをオンにすると変更できます（オフのときは表示のみ）。6ヶ月平均・1年の列の色は、超過（赤）・警告（黄）・以内（緑）です。

        年間（12ヶ月）の範囲

        列を追加 非表示にした列を、ボタンを押すと一覧に戻します。

        36協定 値設定

        社員リストの所属と、下の所属列の文字列が完全一致した行の数値で、1ヶ月（通常・特別）・6ヶ月平均・1年合計を判定します。6ヶ月平均は「法定外・休日労働時間レポート」に複数月（2～6ヶ月平均）列がある場合のみ照合します。編集モードをオンにすると、行の追加・削除と列の削除が使えます（所属列は常に表示されます）。1ヶ月（通常）が1ヶ月（特別）を超えると入力欄が赤、通常が特別の下のしきい値％以上なら黄（特別までの余裕が少ない目安）になります。一覧・Excel の黄信号も同じ％を使います。

        %

        編集モードで変更し、保存すると一覧・Excel に反映されます（既定 80%）。

        バックアップ

        タスク・タスク履歴・社員リスト・メモ・時間外の設定などは、既定ではアプリ専用の保存領域（PC 上のプロファイル内）に保存されます。バックアップ画面のフォルダを選ぶで保存先を指定すると、workhelper-storage.json にも同じ内容のバックアップを自動で書き出せます。別PCへ移すときは JSON のエクスポート、または README の手順に従って更新してください。
